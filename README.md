# vision
运维数据看板

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

> 服务端api提供数据接口 8081
> 页面服务端口 8082
> websocket的端口为8083
> tomcat的端口为8080(默认状态/备用)
